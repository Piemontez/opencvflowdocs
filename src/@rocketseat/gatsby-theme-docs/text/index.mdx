# IAM Controle de Acesso Centralizado

A ferramenta de código aberto (OpenSource) IAM da Thalamus é um sistema SSO (Single Sign-On) que gerencia o acesso centralizado a diversas aplicações.

O IAM é o sistema de autênticação e autorização utilizado em todas as aplicações THALAMUS, 
que pode ser reaproveitado por qualquer empresa que queira centralizar seu sistema de autenticação.
